Patching Cydia on old devices

On older devices, from iOS 1.0 to iOS 7.1.2, Cydia gives a "Unable to Load" error. This is because cydia.saurik.com is signed with the DigiCert Global Root G2 cert, which Apple added to iOS 7.1.2 by default.

This error can be easily fixed, and will get you in the Cydia homepage in no time!

The "Unable to Load" error in Cydia.

Guide

There are 2 different ways to do this.

From iPhoneOS 1.0-iOS 4.3.5

  1. First, go to tlsroot.litten.ca.
  2. Then, press the button that says "Unsigned iOS Bundle".
  3. Press "Install" all the times it tells you.

Cydia should now be patched. You can now use modern repos and see the Cydia homepage.

From iOS 5.0-iOS 7.1.2

  1. First, go to tlsroot.litten.ca.
  2. Then, press the button that says "Signed iOS Bundle (iOS 5+)".
  3. Press "Install" all the times it tells you.

Cydia should now be patched. You can now use modern repos and see the Cydia homepage.
