Patching Cydia on old devices

On older devices, from iOS 1.0 to iOS 7.0.x, Cydia gives a "Unable to Load" error. This is because cydia.saurik.com is signed with the DigiCert Global Root G2 cert, which Apple added to iOS 7.1 by default.

This error can be easily fixed, and will get you in the Cydia homepage in no time!

The "Unable to Load" error in Cydia.

Primary guide

Please note if this guide doesn't work for whatever reason, move to the alternate guide.

  1. Open Cydia, go to the Sources tab, and press Edit.
  2. Type the repo "http://cydia.invoxiplaygames.uk" and add it.
  3. Tap on the new repo called "IPG's Cydia Repo", tap the section called "Tweaks", and find "Cydia HTTPatch".
  4. Tap "Install", then "Confirm", then wait until "Close" appears in the top-right corner.
  5. Cydia should close, and we you re-open it, it should be patched.

If this didn't work, move on to the alternate guide.

Alternate guide

This is an alternate guide, only if the primary guide didn't work.

  1. First, go to tlsroot.litten.ca.
  2. Then, install the certificate called "DigiCert Global Root G2".
  3. Next, install the certificate called "DigiCert Global Root G3".

Cydia should now be patched. It is recommended to install all the certificates to fix more things you may not know about.

You can install all the certs separately, or just use this bundle to install all of them in one go.
