Using Aquila

This guide will help you jailbreak with Aquila.

Aquila is untethered, meaning you only run the exploit once!

Downloads

  • The latest version of Aquila
  • The latest version of iTunes if on Windows

Steps

  1. Plug in your device.
  2. Open the Aquila file.

Aquila should start jailbreaking.
