Using limera1n

Please note limera1n requires a Windows computer running Windows 8 or lower, or MacOS Mojave or lower.

If you're jailbreaking an iPad 1st gen on 3.2.2, we recommend you use greenpois0n, since limera1n is a tethered jailbreak for 3.2.2.

limera1n is capable of jailbreaking A4 devices and the iPhone 3GS and iPod Touch 3rd Generation on iOS 4.0 to 4.1, as well as the iPad 1st Generation on 3.2.2.

Instructions

  1. Extract the zip anywhere. Make sure you can access it!
  2. Open limera1n.
  3. Plug the device in.
  4. Click "make it ra1n".
  5. The device should enter recovery mode.
  6. Follow limera1n's instructions.
  7. When the device has rebooted, open the limera1n app and install Cydia.
  8. Reboot the device again.
